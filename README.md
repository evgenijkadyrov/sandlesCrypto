## Как запустить проект

### 1. Установите зависимости

Убедитесь, что у вас установлен Node.js (версии 18 или выше) и Yarn.

```bash
yarn install
```

## 3 Запустите проект

Запустите проект с помощью команды

```bash
yarn dev
```
